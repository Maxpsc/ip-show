# Privacy Policy for IP Show

IP Show is designed to only query and display your public IP address information. We do not collect, store, or upload any personal data.

## Data We Don't Collect

- We don't use analytics tools
- We don't show advertisements
- We don't track users in any way

## How It Works

This extension queries publicly available IP address information through:

1. **WebRTC STUN Protocol** - Your browser's built-in WebRTC capability is used to detect your egress IP address via STUN servers
2. **Public IP APIs** - We use public IP geolocation APIs to retrieve IP address information

All data is obtained locally from your browser and public APIs. No personal information is transmitted to our servers.

## Third-Party Services

This extension uses the following third-party services:

- Google STUN servers (stun.l.google.com) - For WebRTC IP detection
- Various public IP geolocation APIs - For IP address lookup

These services may log requests according to their own privacy policies.

## Contact

If you have questions about this privacy policy, please open an issue on our GitHub repository.
